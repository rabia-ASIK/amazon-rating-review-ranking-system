# Amazon Rating & Review Ranking System

This project improves product rating and review ranking using:

- Time-Based Weighted Average Rating
- Wilson Lower Bound (WLB) for reliable review ranking

## Project Goal
To calculate a more realistic product rating and identify the most trustworthy reviews in an e-commerce environment.

## Tech Stack
- Python
- pandas
- numpy
- scipy
